# TECH NOTES
Notes about technical elements such as useful commands, so not to forget them in the future and start over again.

## GIT

### Install git from https://git-scm.com/

### Verify git is correctly installed by checking version.
```
git --version
```

### Initialize the repository
```
git init
```

### Configure the user data as global features
```
git config --global user.email <email>
git config --global user.name <name>
```

### Conecting to a specific repository in github
```
git remote add origin <Github repository URL>
```
